## Installation

1. Make sure you have [fastfetch-PNG](https://github.com/Ronin-CK/fastfetch-PNG) installed first.

2. Clone and copy images:
```bash
git clone https://github.com/Ronin-CK/fastfetchPNG-images.git
cp fastfetchPNG-images/*.png ~/.config/fastfetch/images/
```

3. Run `fastfetch` ‚Äî your new images are now in rotation!

## Adding More Images
Drop any PNG with a transparent background into `~/.config/fastfetch/images/`
